## Patch-loading strategy (notebook 03)

Notebook 03 analyses, **for each DDF**, how many distinct Butler
`(tract, patch)` pairs are needed to cover all stable stars in that field.
The parameter `npatchmax` (default: **2**) controls the fallback strategy:

```
n_patches > npatchmax  →  load only band BANDSEL (default: 'r')
n_patches ≤ npatchmax  →  load all 6 bands (u, g, r, i, z, y)
```

This avoids fetching hundreds of large coadd images when a field like COSMOS
spans many patches across the focal plane.

## WCS conversion

The `afw_display.dot()` API works in **pixel** coordinates of the displayed
image.  Conversion from (RA, Dec) to pixels is done via:

```python
from lsst.geom import SpherePoint, degrees
pix = wcs.skyToPixel(SpherePoint(ra_deg * degrees, dec_deg * degrees))
x, y = pix.getX(), pix.getY()
```

where `wcs` is extracted from the loaded `ExposureF` object with
`coadd_exp.getWcs()`.
